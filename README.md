# 🛒 Amazon Clone

A simple Amazon clone built using **HTML, CSS, and JavaScript**.
This project demonstrates product listing, dynamic rendering, and basic cart functionality.

---

## 🚀 Features

* 🛍️ Product listing with images
* ➕ Add to cart functionality
* 🔢 Cart item count update
* 🎨 Responsive UI design
* ⚡ Dynamic rendering using JavaScript

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

```
amazon-clone/
│── index.html
│── style.css
│── script.js
│── images/
```

---

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/yogeshdobliyal/Amazon-clone.git
   ```
2. Open the folder
3. Run `index.html` in your browser
